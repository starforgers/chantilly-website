# Chantilly Website

Chantilly is a romantic, chic, and feminine clothing brand. Every piece is designed with love — crafted to make you feel confident, graceful, and completely yourself. From intricate lace detailing to classy, timeless silhouettes, Chantilly celebrates femininity in its most authentic and refined form.

---

## Stack

- Single `index.html` — no build step, no dependencies
- Tailwind CSS via CDN
- Custom design tokens (beige, red, blue palette)


## Files

| File | Purpose |
|------|---------|
| `index.html` | The page |
| `reference.png` | Design reference image |
| `styles.md` | Design tokens and CSS notes |
| `assets/` | Static assets (images, icons) |

## Running

Open `index.html` directly in a browser — no server needed.
